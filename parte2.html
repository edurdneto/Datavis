O Tableau é um software completo. Além de analisar o dataset e suas peculiaridades, ele já consegue ordenar e filtrar facilmente os dados a serem exibidos nos gráficos, que são excelentes. O Tableau dá muitas opções de gráficos e organizações. Uma pena que o Tableau não possua uma função de incorporar o gráfico na própria página, da mesma forma que ela faz no Tableau public. Existe a opção de exportar, porém a imagem é estática e não permite interações.

Percebe-se o contrário com o uso do Highcharts. A interatividade permite maior exploração e estudo dos gráficos. Porém, para construí-los, não foi fácil: É necessário filtrar e selecionar dados do dataset "na mão", isto é, construindo um software próprio para buscar o que você quer e converte-lo para uma variável do js, tornando assim a tarefa complicada e demorada. Além disso, a possibilidade de gráficos para construir não é a mesma. Veja que o segundo gráfico faltou detalhes (Como a avaliação do Rotten Tomatoes), além das barras não estarem bem posicionadas como ficou no Tableau, visto que foi realizada uma "gambiarra" para colorir pela quantidade de Oscars obtidos.
